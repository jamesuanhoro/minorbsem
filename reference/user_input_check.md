# Check user input function

A function that checks user input for adequacy and fails on inadequate
input.

## Usage

``` r
user_input_check(
  type,
  object_1 = NULL,
  object_2 = NULL,
  object_3 = NULL,
  object_4 = NULL
)
```

## Arguments

- type:

  string for type of check

- object_1:

  Object to check

- object_2:

  Object to check

- object_3:

  Object to check

- object_4:

  Object to check
