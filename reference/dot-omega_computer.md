# Asymptotic variance matrix of lower half vector of correlation matrix

Asymptotic variance matrix of lower half vector of correlation matrix

## Usage

``` r
.omega_computer(r_mat, p_ast, ltri_idxs)
```

## Arguments

- r_mat:

  (matrix) Correlation matrix

- p_ast:

  (positive integer) Dimension of strict lower half vector

- ltri_idxs:

  (array) Indices of strict lower half vector

## Value

Asymptotic variance matrix of strict lower half vector of correlation
matrix.
