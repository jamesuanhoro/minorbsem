# A function to reorder asymptotic variance matrix

A function to reorder asymptotic variance matrix

## Usage

``` r
.fix_acov(acov_mat, old_names, new_names)
```

## Arguments

- acov_mat:

  Major paramters table

- old_names:

  Old names from correlation matrix

- new_names:

  Old names from lavaan object

## Value

Returns reordered acov_mat
